# 产品图片目录说明

## 图片要求

### 产品展示图片
- **主图**: `/images/products/` 目录下的产品图片
- **尺寸**: 建议至少 800×800 像素
- **格式**: JPG/PNG
- **背景**: 白色或透明背景
- **数量**: 每个产品至少3-5张不同角度图片

### 主视觉背景图
- **文件**: `/images/hero-bathroom.jpg`
- **用途**: 产品页面主视觉区背景
- **尺寸**: 1920×1080像素或更高
- **内容**: 展示地漏在真实场景中的应用

### 占位符图片
- **文件**: `/images/placeholder-product.jpg`
- **用途**: 当产品图片加载失败时显示
- **尺寸**: 400×400像素

## 设计规范

### Minimal Industrial 风格
- **主色调**: 黑色、白色、金属灰
- **字体**: 无衬线字体 (如Helvetica, Arial)
- **布局**: 大量留白，清晰的网格系统
- **视觉**: 工业质感，简洁线条

### CRAP 排版原则
1. **对比 (Contrast)**: 标题32px vs 正文16px
2. **重复 (Repetition)**: 统一的卡片和按钮样式
3. **对齐 (Alignment)**: 左对齐为主，严格网格对齐
4. **亲密性 (Proximity)**: 相关信息分组，充足留白

### SEO 优化
- 标题结构清晰 (H1, H2, H3)
- 关键词: "地漏", "floor drain", "不锈钢地漏", "地漏工厂"
- Alt标签完整
- 结构化数据
